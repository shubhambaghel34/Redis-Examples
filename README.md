
# Redis-Examples
Redis examples with nodejs for learning purpose


## Installation
Install Redis with the below command

```bash
npm i redis
```


##Steps to connect Redis
Steps to connect Redis client/server from local

Run redis server


```bash
redis-server
```

To run redis cli

```bash
redis-cli
```
